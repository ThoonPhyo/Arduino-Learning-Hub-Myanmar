Learn Arduino step by step with clear tutorials, pinouts, wiring diagrams, and easy-to-follow code examples.

Introduction

What is Arduino?

Arduino is an open-source electronics platform to build interactive projects with hardware and software.

Getting Started With Arduino Board

Learn what Arduino is, how the board works, and how to begin your first hardware project.

USART in Arduino Uno

Learn serial communication using USART to send and receive data between devices.

PWM in Arduino

Use PWM signals to control motor speed and adjust LED brightness smoothly.

Digital GPIO of Arduino

Understand digital input and output pins to control LEDs, buttons, and basic sensors.

Using Arduino IDE

Arduino IDE is the software used to write, compile, and upload code to Arduino boards.

Software Setup

Explore the features in the IDE

Explore some of the features, functions, and tools that are provided in the IDE.

Using the Menu of the IDE

From the File menu, choose Examples > 01.Basics > Blink. The Blink sketch will open in a new window.

Connecting with the Board

Locate the USB cable. Plug the flat end of the USB cable into your computer. Plug the other end of the cable into the USB port on the Arduino board.

Uploading the Program

In the Arduino IDE, click the Upload button to transfer the code to the Arduino board.

Arduino I2C

I2C (Inter-Integrated Circuit) is a serial bus interface connection protocol. It is also called a TWI (two wire interface) since it uses only two wires for communication.
